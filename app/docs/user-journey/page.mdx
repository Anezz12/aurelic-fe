# User Journey

## Overview

This guide walks through the complete user journey for both liquidity providers and borrowers using Invalend Protocol.

## Liquidity Provider Journey

### Step 1: Connect Wallet
1. Navigate to the Invalend dApp
2. Click "Connect Wallet"
3. Select your preferred wallet (MetaMask, WalletConnect, etc.)
4. Approve the connection

### Step 2: Get Test Tokens
1. Switch to Base Sepolia network
2. Visit the faucet page
3. Request test USDC tokens
4. Wait for tokens to arrive in your wallet

### Step 3: Deposit to Pool
1. Navigate to the Pool page
2. Enter the amount of USDC you want to deposit
3. Click "Approve USDC" and confirm the transaction
4. Click "Deposit" and confirm the transaction
5. Receive Invalend Pool Shares (IPS) representing your deposit

### Step 4: Monitor Performance
1. View your pool share balance
2. Track yield accrual (6% APY simulated)
3. Monitor pool utilization
4. Check your earnings over time

### Step 5: Withdraw Funds
1. Navigate to the Pool page
2. Enter the amount you want to withdraw
3. Click "Withdraw" and confirm the transaction
4. Receive USDC back to your wallet

## Borrower Journey

### Step 1: Connect Wallet
1. Navigate to the Invalend dApp
2. Click "Connect Wallet"
3. Select your preferred wallet
4. Approve the connection

### Step 2: Get Test Tokens
1. Switch to Base Sepolia network
2. Visit the faucet page
3. Request test USDC tokens
4. Wait for tokens to arrive in your wallet

### Step 3: Create Loan
1. Navigate to the Loan page
2. Enter the desired loan amount
3. Review the required collateral (20% of loan amount)
4. Click "Approve USDC" for the collateral amount
5. Click "Create Loan" and confirm the transaction
6. Receive a restricted wallet address for trading

### Step 4: Execute Trades
1. Navigate to the Trading page
2. Select the trading pair (USDC/ETH, USDC/BTC, etc.)
3. Enter the trade amount
4. Set slippage tolerance
5. Click "Execute Swap" and confirm the transaction
6. Monitor your position in real-time

### Step 5: Monitor Position
1. View your restricted wallet balance
2. Track P&L in real-time
3. Monitor health factor
4. Check position status

### Step 6: Close Position
1. Navigate to the Loan page
2. Click "Repay Loan"
3. Confirm the transaction
4. Receive remaining collateral + profits
5. Loan is marked as repaid

## DAO Treasury Journey

### Step 1: Connect DAO Wallet
1. Use a DAO treasury wallet (Gnosis Safe, etc.)
2. Connect to the Invalend dApp
3. Approve the connection

### Step 2: Deploy Capital Efficiently
1. Enter desired position size (e.g., $500K)
2. Review required collateral (20% = $100K)
3. Create loan with treasury funds
4. Receive 5x leverage for deployment

### Step 3: Execute Strategy
1. Use restricted wallet for trading
2. Execute multiple trades as needed
3. Monitor performance in real-time
4. Adjust strategy based on market conditions

### Step 4: Close and Settle
1. Close all positions when strategy is complete
2. Repay loan to pool
3. Receive remaining capital + profits
4. Deploy profits back to treasury

## Troubleshooting

### Common Issues

#### Connection Problems
- Ensure you're on Base Sepolia network
- Check wallet connection status
- Try refreshing the page
- Clear browser cache if needed

#### Transaction Failures
- Check gas fees and limits
- Ensure sufficient token balance
- Verify contract addresses
- Try again with higher gas limit

#### Trading Issues
- Verify restricted wallet has funds
- Check token whitelist status
- Ensure slippage tolerance is appropriate
- Monitor network congestion
