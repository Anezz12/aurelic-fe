# Features

## Core Protocol Features

### 20/80 Prefunding Mechanism

The protocol implements a prefunding model where borrowers provide 20% collateral and the liquidity pool provides 80% funding.

**Functionality:**
- Borrowers deposit 20% collateral (margin)
- Liquidity pool provides 80% funding
- Combined capital enables 5x leverage
- Risk and rewards distributed proportionally

**Benefits:**
- 5x capital efficiency with minimal collateral
- Shared risk and reward structure
- Improved capital utilization compared to traditional lending
- Transparent fund allocation on-chain

**Implementation:**
- Smart contract enforces 20/80 ratio validation
- Automatic fund allocation to restricted wallets
- Real-time tracking of margin and pool contributions
- Proportional profit and loss distribution

### Restricted Smart Wallets

The protocol creates isolated trading environments for each borrower with enforced restrictions on protocol access.

**Functionality:**
- Creates dedicated trading environment for each borrower
- Enforces whitelist-only protocol access
- Prevents unauthorized external transfers
- Enables trading control within defined boundaries

**Benefits:**
- Prevents unauthorized fund drainage
- Maintains borrower control within restrictions
- Eliminates need for trust or human oversight
- Provides transparent execution of all actions

**Implementation:**
- Factory pattern creates unique wallet per borrower
- Whitelist enforcement at smart contract level
- Transaction-level validation for all operations
- Event emission for complete transparency

### ERC-4626 Liquidity Vaults

The protocol implements standardized vaults for liquidity provider deposits using the ERC-4626 standard.

**Functionality:**
- Standardized vault system for LP deposits
- Share-based accounting with yield accrual
- Automated yield distribution (6% APY simulated)
- Compatible with DeFi ecosystem standards

**Benefits:**
- Standardized interface for DeFi compatibility
- Automated yield with principal protection
- Transparent share-based accounting system
- Easy integration with existing DeFi protocols

**Implementation:**
- OpenZeppelin ERC-4626 standard implementation
- Custom yield calculation logic
- Automated share minting and burning
- Real-time yield accrual tracking

**Functionality:**
